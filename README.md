# Future fund - Australia's sovereign wealth fund

Analysis on the Australian sovereign wealth fund: Future Fund


### Funds held under management

This page presents the "Funds held under management" figures, as extracted from [Wikipedia](https://en.wikipedia.org/wiki/Future_Fund). These are understood to be in Australian dollars at the time of reporting, i.e. not adjusted for inflation, benchmark returns etc. The source reports are the Portfolio Updates produced as at 31 December each year.
